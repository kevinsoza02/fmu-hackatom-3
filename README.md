# Banco

## Objetivo

O projeto é uma simulação de banco com conta corrente e features de investimento com IA.
Entrega para o 3 HACKATHON da FMU

## Requisitos

- [Python 3.12](https://www.python.org/downloads/)
- [Venv configurada](https://docs.python.org/3/library/venv.html)
- [Poetry](https://python-poetry.org/docs/)

## Como rodar

1. poetry install
2. python app.py
